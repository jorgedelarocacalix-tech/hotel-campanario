## Archivos

- `admin.html` — panel para que Pahola cambie fotos sin saber código
- `img/bg-img/` — fotos reales: comedor.jpg, eventos.jpg, piscina.jpg + 4 fotos eventos (stock)
- `img/core-img/logo_white.png` — logo oficial del hotel

## Diseño
- Colores: terracota #8B3A1F + dorado #C9A84C + crema #FAF7F2
- Fuentes: Playfair Display + Inter
- Estilo: luxury boutique inspirado en Copantl/Marriott

## Features implementados
- Carrusel hero con drag/swipe (comedor + eventos)
- 4 tarjetas de habitaciones con precios reales
- Sección de eventos completa (bodas/quinceañeros/corporativos/fiestas) con fotos
- Popup de promoción mensual automático (cambia cada mes, código CAMP-MES26)
- Ribbon de promo en la parte superior
- Modal de reservas: Booking.com + Agoda + WhatsApp
- Panel admin para cambiar fotos sin código
- WhatsApp flotante
